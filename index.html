Portfolio

An interactive website built using ASP.NET MVC. I focused on the cast member pages, which included setting up CRUD functionality.

Stained Glass Site

An example homepage for a company who makes and sells stained glass windows. I styled it using CSS and it contains a navbar, image containers and a contact form.

Drive-in Site

A re-design of the site for a local drive-in movie theater. It was styled using Bootstrap 4. It contains many Bootstrap components, including a navbar, jumbotron and cards.

Car Insurance Site

An MVC web application that mimics a car insurance website. It takes user's input on policy information and calculates a quote based on business logic.

Seattle Musicals Site

An example homepage for promoting musicals that will be in Seattle. I styled it using CSS and it contains a navbar, a modal gallery of images and a pop-up contact form.

An application that lets the user keep track of tasks they need to do. It retains the tasks input by the user until removed.

A basic 4 function calculator designed using JavaScript. Users can add, subtract, multiply, and divide.
